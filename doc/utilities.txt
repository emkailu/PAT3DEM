# Utilities here are non-specific but useful programs, for pat3dem

***p3keepconnected.sh
Keep connected with cluster by keeping typing 'bg\n' when the login window is left in background.
1. Install xdotool; Login in a new window; Run this script in a new tab; Go back to the login tab.
2. Whenever you do not use the login tab, put the login window in background.

***p3keepconnectedsshfs.sh
Keep sshfs connected.

***p3findkey.py
Find the files/lines that contain key words.

***p3finddif.py
Find the lines that are different in two files.
